# ShareShot

Screenshot annotation tool. Upload an image, draw on it, download the result.

No server. Runs entirely in the browser — canvas state is saved to localStorage.

## Use it

**https://yuandesu.github.io/shareshot**

Or open `public/index.html` locally in any browser.

## Stack

- Fabric.js canvas (CDN)
- Zero dependencies, zero build step
- GitHub Pages for hosting
